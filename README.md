# Reality Rift

A fast-paced fantasy action game built in **3 days** for **SOP Game Jam**.
https://itch.io/jam/sop-game-jam

## Overview
In the Kingdom of Wizards, one wizard discovers the forbidden spell **"The Infinite"**, granting him limitless magical power. The spell disrupts the balance of the world.

In response, the Master Wizard casts the **Instability Curse** on him, causing his magic to become unstable and shift periodically.

As the player, your mission is to strengthen your magical stability, challenge the Master Wizard, and defeat him to break the curse and restore control over your powers.

## Core Objective
- Increase the stability of your magic over time.
- Survive unstable magical shifts.
- Confront and defeat the Master Wizard.
- Restore stable magical abilities and rebalance the kingdom.

## Development
- **Game Jam:** SOP Game Jam
- **Development Time:** 3 days
- **Engine:** Unity
- **Language:** C#

## Running the Project
1. Open the project folder in Unity Hub.
2. Use a compatible Unity Editor version for the project.
3. Open the main gameplay scene from the `Assets` folder.
4. Press Play in the Unity Editor.

## Team Note
This project was created under game jam constraints, so gameplay-first implementation and rapid iteration were prioritized.
